torch_geometric.transforms¶

Transforms are common torch_geometric.datasets.Data transforms for graph datasets. They can be chained using the Compose transform from PyTorch.

CartesianAdj¶

class torch_geometric.transforms.CartesianAdj[source]¶

Converts a set of adjacency matrices \(A \in \mathbb{R}^{N x N}\) saved in a dataset Data object to globally normalized Cartesian pseudo-coordinates \(A \in [0, 1]^{N x N x d}\) based on the position \(P \in \mathbb{R}^{N x d}\) of graph-nodes.

PolarAdj¶

class torch_geometric.transforms.PolarAdj[source]¶

Converts a set of adjacency matrices \(A \in \mathbb{R}^{N x N}\) saved in a dataset Data object to globally normalized polar pseudo-coordinates \(A \in [0, 1]^{N x N x 2}\) based on the position \(P \in \mathbb{R}^{N x 2}\) of graph-nodes.

SphericalAdj¶

class torch_geometric.transforms.SphericalAdj[source]¶

Converts a set of adjacency matrices \(A \in \mathbb{R}^{N x N}\) saved in a dataset Data object to globally normalized spherical pseudo-coordinates \(A \in [0, 1]^{N x N x 3}\) based on the position \(P \in \mathbb{R}^{N x 3}\) of graph-nodes.

TargetIndegreeAdj¶

class torch_geometric.transforms.TargetIndegreeAdj[source]¶

Converts a set of adjacency matrices \(A \in \mathbb{R}^{N x N}\) saved in a dataset Data object to the globally normalized indegree of the target nodes \(u(i, j) = \deg(j)/\max_{i \in \mathcal{V}}(\deg(i))\).

Graclus¶

class torch_geometric.transforms.Graclus(level=1)[source]¶

Converts an adjacency matrix to level coarsened graphs saved in data['adjs'] based on the “Convolutional Neural Networks on Graphs with Fast Localized Spectral Filtering” paper.

Parameters:level (int) – Number of coarsened graphs to compute. (default: 1)

Graclus and pseudo-coordinate transforms can be composed:

Example

>>> Compose([Graclus(4), CartesianAdj()])

Single Example Data¶

class torch_geometric.datasets.Data(input, adj, position, target)[source]¶

Single Example Dataholder Object that can hold input feature matrices, adj sparse matrices, graph-node position and target information. Additional properties can be added manually.

add(key, value)[source]¶

Adds a property as key-value pair.

all()[source]¶

Returns all data properties as dict.

FAUST¶

class torch_geometric.datasets.FAUST(root, train=True, transform=None)[source]¶

MPI FAUST Dataset.

Parameters:
  • root (string) – Root directory of dataset where raw/MPI-FAUST exist. Dataset needs to be downloaded manually.
  • train (bool, optional) – If True, creates dataset from training.pt, otherwise from test.pt. (default: True)
  • transform (callable, optional) – A function/transform that takes in a Data object and returns a transformed version. (default: None)

MNISTSuperpixels¶

class torch_geometric.datasets.MNISTSuperpixels(root, train=True, transform=None)[source]¶

MNIST Superpixels Dataset from the “Geometric Deep Learning on Graphs and Manifolds Using Mixture Model CNNs” paper. Each example contains exactly 75 graph-nodes.

Parameters:
  • root (string) – Root directory of dataset. Downloads data automatically if dataset doesn’t exist.
  • train (bool, optional) – If True, creates dataset from training.pt, otherwise from test.pt. (default: True)
  • transform (callable, optional) – A function/transform that takes in a Data object and returns a transformed version. (default: None)
